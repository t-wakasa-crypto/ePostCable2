# scripts ディレクトリ

開発作業を補助するシェルスクリプト群です。ファイル名は `<カテゴリ>-<操作>.sh` の形式です。

## 概要

スクリプト一覧・各スクリプトの説明・追加修正のルールは本ファイルではなく
[docs/システム開発スクリプト設計書.md](../docs/システム開発スクリプト設計書.md)
に記載する。**スクリプトを使う・追加する・修正する前に、必ず同書を Read し、
そこに記載された一覧・命名規則に従うこと。** スクリプト一覧は
`./scripts/help.sh` でも確認できる。

## 更新ルール

スクリプトを新規作成・修正した場合は、**必ず
[docs/システム開発スクリプト設計書.md](../docs/システム開発スクリプト設計書.md)
と `scripts/help.sh` の一覧を同時に更新すること**。本ファイル（薄いポインタ）は、
参照先が変わる場合を除き更新不要。

## 禁止事項
- `docs/システム開発スクリプト設計書.md` を読まずにスクリプトを追加・修正すること
- スクリプト一覧・ルールを本ファイルに書き足し、
  `docs/システム開発スクリプト設計書.md` と内容が二重管理・不整合になること
- スクリプトを追加・修正したのに `docs/システム開発スクリプト設計書.md` や
  `help.sh` を更新しないまま放置すること
